# Statistics (spending insights)

## Category breakdown

Pie chart with the different categories of spending.

Data format:

```json
[
  {
    "category": "food",
    "spending": 301.82
  },
  {
    "category": "clothing",
    "spending": 87.05
  }
]
```

## Spending over time

Line/bar chart with the amount spent each day

Data format:

```json
[
  {
    "date": "2020-01-18",
    "spending": 32.55
  },
  {
    "date": "2020-04-02",
    "spending": 67.99
  }
]
```
